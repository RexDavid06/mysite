CHAT API

   

I developed a chat API that enables real-time messaging. It uses JWT (JSON Web Token) authentication to ensure that only authorized users can send or receive messages. Upon login, users receive a token that is used in subsequent requests to verify their identity. The API supports various features such as sending messages, viewing chat history, and managing user sessions, all while enforcing role-based access control. The token-based system ensures that sensitive user data is protected and that interactions within the chat platform remain secure.
